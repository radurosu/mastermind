Solver is a faithful one shot prompt Fable-generated JavaScript port of mmSolver.java (1997), done on 7/9/2026: shuffle the full code set once, then each turn submit the first code in the shuffled order that is consistent with every previous (guess, result) pair. The thesis measured an average of 4.64 guesses and a worst case of 8 for the classic {6 colors, 4 positions} game, and showed near-linear growth of expected guesses across code spaces from {2,1} to {12,8}. Radu T. Rosu, “Analysis of the game of Mastermind — the mn case,” Undergraduate Honors Thesis, North Carolina State University, Dr. Donald L. Bitzer (chair), Dec 10 1997.
