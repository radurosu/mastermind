Conclusions
 
 
            By looking at both the heuristic and deep search algorithm for the game of mastermind, one can see the performance of the heuristic algorithm over time.  The results achieved lead to the conclusion that the heuristic algorithm gives very good results  with any (color,position) combination and that it is the algorithm of choice.
